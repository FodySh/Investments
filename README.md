# 📊 محفظتي المالية

تطبيق ويب شخصي لإدارة المحفظة الاستثمارية — يعمل مباشرة من المتصفح بدون خادم.

---

## ✨ المميزات

### 🏠 لوحة التحكم
- نظرة شاملة على المحفظة مع KPI cards (إجمالي مستثمر، عوائد، ROI)
- مخطط دائري لتوزيع الأصول (صكوك، صناديق، أسهم)
- أشرطة توزيع حسب المنصة
- توزيع المخاطر حسب السنة
- فلتر بالسنة
- أفضل الاستثمارات أداءً

### 📈 الأسهم
- تتبع الأسهم السعودية والأمريكية في نفس الواجهة
- جلب الأسعار الحية عبر Cloudflare Worker
- حساب الربح/الخسارة غير المحققة تلقائياً
- فلتر بالمنصة والحالة والبحث
- تسجيل توزيعات الأرباح وتتبعها
- تصدير واستيراد Excel

### 🏦 الفرص الاستثمارية
- إدارة الصكوك والمرابحات والصناديق والمعادن الثمينة
- نموذج إضافة متعدد الخطوات مع حسابات تلقائية (ROI، MOIC، أيام متبقية)
- تعديل سريع وتعديل كامل مباشرة من الجدول
- فلتر بالسنة والحالة والبحث
- تصدير واستيراد CSV
- تسجيل دفعات العوائد وتتبع الإيرادات

### 🏦 القروض البنكية
- إضافة قروض مع حساب PMT تلقائي
- جدول سداد تفصيلي (Amortization Schedule)
- تتبع الأقساط المدفوعة والمتبقية
- جدول أقساط متعددة لقروض ذات أقساط متغيرة
- تصدير جدول السداد CSV

### 📊 التقارير والتدفق المالي
- مخطط التدفق النقدي الشهري (داخل/خارج/صافي)
- KPIs سنوية
- مصادر الدخل والإنفاق المجمّعة

### 🗄️ قاعدة البيانات
- جدول spreadsheet قابل للتعديل المباشر
- تبويبات للأسهم والصكوك والصناديق
- بحث وفلترة
- تعديل وتكرار وحذف مع undo
- تصدير JSON واستيراد

## ⚙️ الإعداد

### Firebase
1. أنشئ مشروع على [Firebase Console](https://console.firebase.google.com)
2. فعّل Authentication (Google provider)
3. فعّل Firestore Database
4. ضع `firebaseConfig` في `index.html`

**قواعد Firestore المطلوبة:**
```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
```

### Cloudflare Worker (اختياري)
مطلوب لجلب أسعار الأسهم وتشغيل AI Analyst. انشر Worker بسيط يعمل كـ proxy لـ Yahoo Finance وـ Anthropic API، ثم ضع الرابط في متغير `PROXY_URL`.
