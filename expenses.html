⚡ الطريقة الأولى — Automation تلقائي (ينفّذ عند وصول رسالة البنك)
1.افتح تطبيق Shortcuts على آيفون
2.اضغط تبويب Automation من الأسفل ← اضغط +
3.اختر Message ← في From اكتب اسم البنك كما يظهر في الرسالة (مثال: ALINMA أو SNB)
4.اضغط Next ← New Blank Automation
5.اضغط Add Action ← ابحث Open URLs ← اختره
6.الصق في URL: https://fodysh.github.io/expenses.html#add
7.اضغط Done ← أوقف Ask Before Running ← احفظ
🏠 الطريقة الثانية — أيقونة على الشاشة (أسرع إعداداً)
1.افتح Safari وانتقل للرابط: https://fodysh.github.io/expenses.html#add
2.اضغط زر المشاركة 🔗 في أسفل Safari
3.اختر Add to Home Screen ← سمّها تسجيل مصروف ← Add
